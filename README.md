# Panz Auto

Landing page performance brand untuk `Panz Auto` dengan visual 3D, animasi GSAP, dan struktur proyek modern berbasis Vite.

## Stack

- Vite
- Vanilla JavaScript
- Three.js
- GSAP
- Lenis

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Struktur

```text
.
|- index.html
|- public/
|  |- images/
|  |- models/
|- src/
|  |- main.js
|  |- styles/
|     |- main.css
|- Panz Auto.html
|- package.json
|- vite.config.js
```

`Panz Auto.html` disimpan sebagai file referensi konsep awal.
